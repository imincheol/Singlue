# Singlue Documentation

**Singlue** 프로젝트의 공식 문서 인덱스입니다.
이 프로젝트는 **문서 기반 개발 (Documentation Driven Development)** 방법론을 따릅니다.
개발 전 문서를 먼저 작성/수정하고, 이를 바탕으로 코드를 구현하십시오.

- **[규칙 (Rules)](./rules.md)**: 개발 프로세스, 원본 문서 보존 규칙, 코딩 컨벤션.
- **[기술 스택 (Tech Stack)](./tech_stack.md)**: 사용된 기술, 아키텍처, 디렉토리 구조.

### 2. 스펙 및 기능 (Specs & Features)
- **[정보 구조 (Data Structure)](./data_structure.md)**: Song, Lyrics, VideoMapping 등 데이터 모델 정의.
- **[구현된 기능 (Features)](./features.md)**: 현재 구현 완료된 기능 상세 명세.

### 3. 로드맵 (Roadmap)
- **[백로그 (Backlog)](./backlog.md)**: 미개발 사항, 임시 구현체, 개선 필요 항목, To-Do 리스트.
